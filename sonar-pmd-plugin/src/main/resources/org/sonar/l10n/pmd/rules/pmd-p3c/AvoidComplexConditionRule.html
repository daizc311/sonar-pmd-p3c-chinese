不要在条件语句中使用复杂的语句 (除了常用的方法，如 getXxx/isXxx). 使用boolean变量临时存储复杂语句的结果会增加代码的可读性。

Examples:

    Negative example:
    if ((file.open(fileName, "w") != null) && (...) || (...)) {
        // ...
    }

    Positive example:
    boolean existed = (file.open(fileName, "w") != null) && (...) || (...);
    if (existed) {
        //...
    }
