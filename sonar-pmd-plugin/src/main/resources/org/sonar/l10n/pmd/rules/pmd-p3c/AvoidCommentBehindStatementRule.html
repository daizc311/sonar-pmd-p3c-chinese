方法中的单行注释应该放在要注释的代码上方，单行使用 // ，多行使用/* */。
Examples: 
    public void method() {
        // Put single line comment above code. (Note: align '//' comment with code)
        int a = 3;

        /**
        * Some description about follow code. (Note: align '/**' comment with code)
        */
        int b = 4;
    }
