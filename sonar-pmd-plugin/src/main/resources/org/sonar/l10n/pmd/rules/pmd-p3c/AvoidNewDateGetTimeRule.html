使用 System.currentTimeMillis() 获取当前毫秒。不要使用新的 Date().getTime()。
注意：为了获得更准确的时间，请使用 System.nanoTime()。在 JDK8 中，使用 Instant 类来处理时间统计等情况。 Examples: 
    public class TimeMillisDemo {
        public static void main(String args[]) {
            // Positive example:
            long a = System.currentTimeMillis();
            // Negative example:
            long b = new Date().getTime();

            System.out.println(a);
            System.out.println(b);
        }
    }
